# L'Ancre Têxtil - Site Institucional Premium

Projeto front-end completo desenvolvido com as melhores práticas de semântica e performance utilizando tecnologias web nativas.

## 🚀 Tecnologias Utilizadas
- **HTML5 Semântico**
- **CSS3 Estruturado** (Layout responsivo via CSS Grid e Flexbox)
- **Vanilla JavaScript** (Sem dependência de bibliotecas ou frameworks)

## 📁 Como Executar o Projeto Localmente
1. Monte a árvore de diretórios conforme ilustrado na estrutura do projeto.
2. Certifique-se de salvar os arquivos CSS na pasta `css` e o script na pasta `js`.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

## 🌐 Publicação Gratuita (GitHub Pages)
1. Crie um repositório no GitHub e envie os arquivos mantendo a estrutura de pastas.
2. Vá em **Settings** > **Pages**.
3. Em *Source*, selecione a branch `main` (ou correspondente) e a pasta raiz `/`.
4. Salve. Em instantes, sua URL de visualização estará pronta.

## 🛠️ Manutenção do Catálogo
- **Adicionar Produtos**: Duplique a estrutura `<div class="product-card">` localizada dentro da seção `#produtos` no arquivo `index.html` e edite os metadados.
- **Substituir Imagens**: Modifique o atributo `src` dos cards de produto ou os atributos `data-src` da galeria para apontar para as suas imagens locais salvas na pasta `images/`.
